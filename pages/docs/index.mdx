## What is Sew Protocol?

Sew Protocol lets applications route ERC-20 transfers through non-custodial escrow, with predefined release and dispute resolution paths. It provides a standard primitive for:

- **Protected Transfers**: ERC-20 transfers held under rule-bound conditions.
- **Deterministic Settlement**: Release and resolution following predefined transitions.
- **Per-Escrow Isolation**: Independent agreements where failures cannot propagate.
- **Forward-Only Evolution**: Updates that apply only to new escrows, preserving active ones.
