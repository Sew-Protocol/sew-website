## What is Sew Protocol?

Sew Protocol is a non-custodial escrow architecture for ERC20 transfers on Ethereum. It provides:

- **Protected Transfers**: Funds are held in escrow until release conditions are met
- **Deterministic Settlement**: Release and resolution follow predefined rules
- **Per-Escrow Isolation**: Each agreement is independent; failures don't propagate
- **Forward-Only Upgrades**: Changes affect new escrows only, preserving existing agreements
